Increase Anki Latex Resolution

  1. Using the add-on code available in Edit LaTeX and dvipng calls, in Anki go to Tools -> Addons -> Browse and Install, and enter the code.
  2. Restart Anki.
  3. In Anki, go to Tools -> Addons -> Edit LaTeX and dvipng calls -> Edit, which will open a text file.
  4. Change the numeral element of the array assingned to newDviPng from 200 (default) to 300, or a larger value. This determines the resolution of the rendered latex image.
