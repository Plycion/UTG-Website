Regenerate Anki Latex Cards

  1. Delete old latex files: Navigate your file system to the folder "Anki" which is located somewhere in your home folder (under OS X it is located in "Documents").
  2. In this folder, navigate to the folder "your_username", which by default is "User 1".
  3. In this folder you will find a folder "collections.media". Empty this folder (delete all its files).
  4. In Anki, go to Tools -> Maintenance -> Unused Media. This will regenerate all the Latex files.
